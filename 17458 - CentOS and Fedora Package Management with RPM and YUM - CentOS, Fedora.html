Red Hat Package Manager (RPM) and Yellowdog Updater, Modified (YUM) are open source and default package management utility tools for Red Hat based operating system like CentOS and Fedora. Using RPM and YUM, administrator can perform install, reinstall, uninstall, update and upgrade packages. RPM and YUM are also used to check the version, status, signature and dependencies using the RPM and YUM tools. A package consists of an archive of files and meta-data used to install and erase the archive files.

This article will guide you for package management using RPM and YUM respectively.

RPM

The RPM formerly known as .rpm file, includes compiled software programs and libraries needed by the packages. This utility only works with packages that are built on .rpm format.

Check the RPM signature package. rpm -checksig http://vault.centos.org/centos/7/extras/Source/SPackages/python-sqlalchemy-0.9.8-1.el7.src.rpm
Install the RPM package. sudo rpm -ivh http://vault.centos.org/centos/7/extras/Source/SPackages/python-sqlalchemy-0.9.8-1.el7.src.rpm
-i: Install a package.
-v: Verbose the message/output.
-h: Print the hash of the packages.
Check dependencies for installation. sudo rpm -qpR http://vault.centos.org/centos/7/extras/Source/SPackages/python-sqlalchemy-0.9.8-1.el7.src.rpm
Option description:
-q: Query a package.
-p: List capabilities this package provides.
-R: List the package dependencies.
Install a RPM Package without dependencies. sudo rpm -ivh --nodeps http://vault.centos.org/centos/7/extras/Source/SPackages/python-sqlalchemy-0.9.8-1.el7.src.rpm
List the files installed by RPM. sudo rpm -ql python
List the recently installed RPM packages. sudo rpm -qa -last
Print all the installed packages. sudo rpm -qa
Upgrade the RPM packages. sudo rpm -Uvh --nodeps ftp://rpmfind.net/linux/centos/6.7/os/x86_64/Packages/httpd-2.2.15-45.el6.centos.x86_64.rpm
Uninstall an RPM package. sudo rpm -evv httpd
Uninstall the RPM package without dependencies. sudo rpm -ev --nodeps httpd
Query a file that belongs to the RPM package. sudo rpm -qf /usr/bin/htpasswd
Print the information of installed RPM. sudo rpm -qi httpd
Verify the installed RPM package. sudo rpm -Vp ftp://rpmfind.net/linux/centos/6.7/os/x86_64/Packages/httpd-2.2.15-45.el6.centos.x86_64.rpm
Import the PGP keys to verify the packages using the following command: sudo rpm --import /etc/pki/rpm-gpg/RPM-GPG-KEY-CentOS-7
List all the imported GPG keys. sudo rpm -qa gpg-pubkey*

YUM

YUM is an automatic update and package management utility. YUM installs all the dependencies automatically.

Update the RPM packages. sudo yum update
Update the installed RPM package. sudo yum update httpd
Install a package. sudo yum install httpd
Search a package from the yum repository. sudo yum search nginx
Remove or uninstall the package. sudo yum erase nginx
or
sudo yum remove nginx
Clear the caches of packages. yum clean all
Check if the packages need to be updated. sudo yum check-update
List a package or groups of packages. sudo yum list #to display all packages.
sudo yum list httpd #list selected package
Print what a package does. yum provides httpd
Find the list of RPM dependency. sudo yum deplist package-name
Print the list of groups. sudo yum grouplist | more
Yum group install. sudo yum groupinstall 'Development Tools'
Remove or uninstall YUM group. sudo yum groupremove 'Development Tools'
Update YUM group. sudo yum groupupdate 'Development Tools'
Get YUM group information. sudo yum groupinfo 'Development Tools'
Print the list of installed package in paging. sudo yum list installed | more
