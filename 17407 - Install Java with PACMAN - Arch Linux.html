Install Java

Install Java with pacman - Arch Linux

Difficulty: 1
Time: 10 minutes

Java is used by many applications so behooves many administrators to have Java already installed. This article contains directions for installing both the java runtime, SDK, and Open JDK 7 versions

Install Java

  1. Update your repository
    sudo pacman -Syu
  2. Download Java 8 runtime (update the version number based on the current version)
    sudo pacman -S jre8-openjdk
  3. Download Java OPEN JDK 8 (update the version number based on the current version)
    sudo pacman -S jdk8-openjdk
  4. Create / edit a file, /etc/environment
    sudo vim /etc/environment
  5. Select your Java_home (in this case we will use the jdk version).
    JAVA_HOME="/usr/lib/jdk8-openjdk"
  6. Save and close.
  7. Load the environment file
    source /etc/environment
  8. Test it
    echo $JAVA_HOME

Conclusion

You have just installed Java.
