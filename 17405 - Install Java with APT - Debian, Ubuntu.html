Install Java with apt

Install Java with apt - Debian, Ubuntu

Difficulty: 1
Time: 10 minutes

Java is used by many applications, so it behooves many administrators to have Java already installed. This article contains instructions for installing the Java runtime SDK and Open JDK 7 versions.

Install Java

  1. Update your repository.
    sudo apt-get update
  2. Make a directory for Java.
    sudo mkdir /usr/java
  3. Download Java runtime.
    sudo apt-get install default-jre
    Enter y at the prompt to continue installation.
  4. Download Java SDK.
    sudo apt-get install default-jdk
    Enter y at the prompt to continue installation.
  5. Download Java OPEN JDK 7.
    sudo apt-get -y install openjdk-7-jdk
  6. Make a symbolic link to the Java directory (substitute i386 for amd64 if you are on a 32 bi system).
    sudo ln -s /usr/lib/jvm/java-7-openjdk-amd64 /usr/java/default

Check your Java version

Check the versions of Java you have installed on your server.

java -version

If your output shows that you have just one version of Java installed, like in this example:

java version "1.7.0_79"
OpenJDK Runtime Environment (IcedTea 2.5.6) (7u79-2.5.6-0ubuntu1.14.04.1)
OpenJDK 64-Bit Server VM (build 24.79-b02, mixed mode)

Then you are all done!

If you have more than one Java version installed, continue on where you'll select your default version and set the environment.

Optional: Select and verify the default Java version

If you have multiple versions of Java on the same server, choose your default version.

  1. Select the default version of Java.
    sudo update-alternatives --config java
    You'll see this: 
      Selection    Path                                            Priority   Status
------------------------------------------------------------
* 0            /usr/lib/jvm/java-6-openjdk-amd64/jre/bin/java   1061      auto mode
  1            /usr/lib/jvm/java-6-openjdk-amd64/jre/bin/java   1061      manual mode
  2            /usr/lib/jvm/java-7-openjdk-amd64/jre/bin/java   1051      manual mode

Press enter to keep the current choice[*], or type selection number:
  2. Make a selection (We'll keep it set to 0 for simplicity).

Set your environment

Also important when you have multiple Java versions is setting your environment. Let's do that now.

  1. Create a file, /etc/environment.
    sudo vim /etc/environment
  2. Select your Java_home by trimming jre/bin/java off the path of your preferred alternative chosen above and entering it into the file.
    JAVA_HOME="/usr/lib/jvm/java-6-openjdk-amd64"
  3. Save and close the file.
  4. Load the environment file.
    source /etc/environment
  5. Test it.
    echo $JAVA_HOME

Conclusion

Congratulations! You have succesfully installed Java on your server.
