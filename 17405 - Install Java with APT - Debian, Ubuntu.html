Install Java

Install Java with apt - Debian, Ubuntu

Difficulty: 1
Time: 10 minutes

Java is used by many applications so behooves many administrators to have Java already installed. This article contains directions for installing both the java runtime, SDK, and Open JDK 7 versions

Install Java

  1. Update your repository
    sudo apt-get update
  2. Make a directory for java
    sudo mkdir /usr/java
  3. Download Java runtime
    sudo apt-get install default-jre
  4. Download Java SDK
    sudo apt-get install default-jdk
  5. Download Java OPEN JDK 7
    sudo apt-get -y install openjdk-7-jdk
  6. Make a symbolic link to the Java directory (substitute i386 for amd64 if you are on a 32 bi system.
    sudo ln -s /usr/lib/jvm/java-7-openjdk-amd64 /usr/java/default

Select and verify the default Java version

  1. Select the default version of Java
    sudo update-alternatives --config java
    You will see this: 
      Selection    Path                                            Priority   Status
------------------------------------------------------------
* 0            /usr/lib/jvm/java-6-openjdk-amd64/jre/bin/java   1061      auto mode
  1            /usr/lib/jvm/java-6-openjdk-amd64/jre/bin/java   1061      manual mode
  2            /usr/lib/jvm/java-7-openjdk-amd64/jre/bin/java   1051      manual mode

Press enter to keep the current choice[*], or type selection number:
    Make a selection (we will keep it set to 0 for simplicity).
  2. Create a file, /etc/environment
    sudo vim /etc/environment
  3. Select your Java_home by trimming off the jre/bin/java part of the path of your preferred alternative that you chose above, and entering it into the file.
    JAVA_HOME="/usr/lib/jvm/java-6-openjdk-amd64"
  4. Save and close.
  5. Load the environment file
    source /etc/environment
  6. Test it
    echo $JAVA_HOME

Conclusion

You have just installed three versions of Java and selected one to be your default.
