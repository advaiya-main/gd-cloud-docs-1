Install and Configure PostgreSQL - Debian

Difficulty: 1
Time: 15 minutes

This article describes the installation and configuration of PostgreSQL on Debian. PostgreSQL is an open-source relational database management system. PostgreSQL uses the Structured Query Language (SQL) to manipulate data.

The tasks in this article assume that you have sudo privileges.

Install PostgreSQL

  1. Update all the system packages.
    sudo apt-get update
  2. Install the PostgreSQL package and necessary dependencies.
    sudo apt-get install postgresql postgresql-contrib
  3. Configure PostgreSQL to start up on server boot.
    sudo update-rc.d postgresql enable
  4. Start the PostgreSQL service.
    sudo service postgresql start
    Once PostgreSQL installs completely, the postgres system user and the database are automatically created.

Connect to PostgreSQL

  1. Log in with the postgres user.
    sudo su - postgres
  2. Log into the database.
    Psql
  3. Exit from the Postgres=# prompt.
    \q
  4. Log out the postgres system user.
    exit
