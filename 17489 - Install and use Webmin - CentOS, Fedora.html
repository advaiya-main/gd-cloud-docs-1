Install Webmin - CentOS, Fedora

Difficulty: 2
Time: 15

Webmin is a web-based tool for managing your server. It supports many common tasks like installing software, adding users, and managing servers. This article describes how to install webmin. It assumes you have a user, yourloginuser, who accesses your server via SSH.

Create a user with sudo privileges

  1. Edit the /etc/sudoers file with visudo
    sudo visudo
  2. Find the section that looks like this: 
    root    ALL=(ALL) ALL
  3. Add a line in it for the user you use to log in to your server to grant sudo privileges to that user. 
    root    ALL=(ALL) ALL 
yourloginuser    ALL=(ALL) ALL
  4. Save and close

Install Webmin

  1. Create a webmin repository file
    sudo vim /etc/yum.repos.d/webmin.repo
  2. Add the following lines to the file: 
    [Webmin]
name=Webmin Distribution Neutral
#baseurl=http://download.webmin.com/download/yum
mirrorlist=http://download.webmin.com/download/yum/mirrorlist
enabled=1
  3. Save and close
  4. Add the Webmin key to your keyring
    sudo wget http://www.webmin.com/jcameron-key.asc
  5. Import the public key
    sudo rpm --import jcameron-key.asc
  6. Update your local repository
    sudo yum update
  7. Install webmin
    sudo yum install webmin

Verification

  1. Using your browser, access the Webmin login page:
    https://your IP address:10000
  2. Ignore the SSL warning (you trust this server because you own it.
  3. A login screen will appear. Log in using the yourloginuser and start using webmin!
