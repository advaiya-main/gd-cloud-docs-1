Install and configure Openfire XMPP Server – CentOS, Fedora

Difficulty: 2
Time: 0.5 hours

This article will cover details on how to install and configure Openfire XMPP Server on Ubuntu and Debain.

XMPP – A quick overview

Extensible Messaging and Presence Protocol (XMPP) is a communication protocol for middleware based on XML. It enables the near-real-time exchange of structured yet extensible data between any two or more network entities.

Openfire – A quick overview

Openfire, the real time collaboration (RTC) server, is licensed under the Open Source Apache License. It leverages the XMPP protocol for communication. It is easy to setup and configure, and also provides a web-based interface for administrative activities and management. It offers high level of security and performance.

Openfire do not support chat. An additional client is needed, in similar manner as web servers need a browser.

Install Openfire

  1. Update the operating system
    sudo yum –y update
  2. Install Java
    sudo yum install java
  3. Download Openfire
    sudo wget
    http://www.igniterealtime.org/downloadServlet?filename=openfire/openfire-3.9.3-1.i386.rpm -O /root/openfire-3.9.3-1.i386.rpm
  4. Install Openfire
    sudo yum install -y /root/openfire-3.9.3-1.i386.rpm

    OpenFire package does not require separate installation of Java Runtime Environment (JRE) as it provides its own JRE Environment. This package is available in 32 bit version only. So, in case you are running this installation on a 64 bit machine, corresponding 32 bit libraries must be installed separately

  5. Install 32 bit libraries (in case you are using 64 bit system).
    sudo yum install -y glibc.i686
  6. Start the service again after every reboot
    sudo chkconfig openfire on
  7. Start the service
    sudo systemctl start openfire.service

Test Openfire

  1. Test the Openfire installation on your system, by opening the following URL on your browser
    http://<your_IP>/9090
  2. Successful installation of Openfire would show the Openfire wizard on your browser. Follow the instructions to proceed with the setup.
  3. On the first page, choose the language.
  4. On the second page, provide appropriate values for domain and associated ports
  5. Leave the default values for the next steps.
  6. Once completed with the setup procedure, login into the admin console using the Admin account.
  7. Once logged-in, the first screen shows the server manager tab under the Server tab. The server manager provides information about server properties and environment, system properties, language and time, clustering, database, logs, email setting and security audit viewer.
  8. The server setting tab provides information about profile settings, client connections, server to server connections, external components, http bindings, update management, registration and login management, resources policies, offline messages, audit policies for messages, private data storage options, security settings, server certificates, compression settings and file transfer settings.
  9. The Media services tab provides information about Media Proxy settings.

Conclusion

By using all of the above steps, you will be able to install and configure openfire XMPP server on CentOS and Fedora.
