Install Ruby on Rails using RVM - Debian

Difficulty: 1

Time: 30 minutes

Ruby on Rails is a well-known open source stack for web application development. With the Ruby Version Manager, you can easily manage multiple Ruby application environments and switch between them in the most efficient way. This article will assist on how to install Ruby on Rails in Debain VPS using RVM.

You will need access to sudo privileges to complete these steps.

Install RVM

  1. Update all the packages of operating system using the following command:
    sudo apt-get -y update
  2. Install RVM using the following commands:
    sudo apt-get install libgdbm-dev libncurses5-dev automake libtool bison libffi-dev
    sudo curl -L https://get.rvm.io | bash -s stable

    when you run the above command, then there is one dependency, so fix the dependency and run the above command again.

  3. Run the following commands to fix dependencies:
    sudo curl -sSL https://rvm.io/mpapis.asc | gpg --import -
    sudo curl -L https://get.rvm.io | bash -s stable
  4. Reload rvm profile to enable rvm command:
    source ~/.rvm/scripts/rvm
  5. To install Ruby and check Ruby version using following command:
    rvm install ruby
    ruby -v
  6. Ruby tends to install the contents for all the packages. Type the following command to inhibit this:
    sudo echo "gem: --no-ri --no-rdoc" > ~/.gemrc
  7. Install the bundler.
    gem install bundler
  8. To prevent any dependencies of Rails libraries, install nodejs by typing:
    sudo add-apt-repository ppa:chris-lea/node.js
    sudo apt-get update
    sudo apt-get install nodejs

Install Rails

  1. Run the following command to install Rails:
    gem install rails

    This will take some time to create the environment.

  2. Type the following command to check you Rails version:
    rails -v
  3. Create the project directory for your Rail project by typing the following command:
    sudo mkdir â€“p /usr/local/src/rail_project
  4. Move to the newly created directory and grant all the required access using the following commands:
    cd /usr/local/src/rail_project
    sudo chmod -R 777 /usr/local/src/rail_project
  5. Create a Rails application and go to the application directory.
    rails new Your_application
    cd Your_application
  6. Ensure the Rails application is running on Rails server.
    rails server
  7. Bind the Rails application using the following command:
    rails server --binding=[your public IP]

The new application is ready and you can access it by visiting your_public_IP_or_domain:3000 (3000 is default port).

Conclusion

In this article, you learned the steps involved in setting up Ruby on Rails using RVM for the web application on Debian distribution.
