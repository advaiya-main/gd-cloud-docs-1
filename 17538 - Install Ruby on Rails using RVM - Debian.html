Install Ruby on Rails using RVM - Debian

Difficulty: 1

Time: 30 minutes

Ruby on Rails™ is a well-known framework for web application development. The Ruby Version Manager (RVM) allows developers to easily manage multiple Ruby application environments and switch between them efficiently.

This article will assist you on how to install Ruby on Rails in Debian using RVM, and assumes that you have sudo privileges to perform this task. You may want to visit the Initial Server Setup - Debian article to setup your Debian server.

Prepare your system

Before you install Ruby on Rails, prepare your system with the required system software.
  1. Update the system software and access root.
    sudo apt-get update
  2. Install curl.
    sudo apt-get install curl
  3. Install tar.
    sudo apt-get install tar
  4. Install all the dependencies.
    sudo apt-get install libgdbm-dev libncurses5-dev automake libtool bison libffi-dev

Update and install RVM and Ruby

  1. If your system doesn't have RVM already installed, type the following command to install RVM:
    sudo curl -L https://get.rvm.io | bash -s stable
  2. After you run the above command, there are more dependencies that needs to be fixed, so run the following commands:
    sudo curl -sSL https://rvm.io/mpapis.asc | gpg --import -
    sudo curl -L https://get.rvm.io | bash -s stable
  3. To apply RVM, you need to execute the following command:
    source ~/.rvm/scripts/rvm
  4. Install Ruby and check Ruby version using following commands:
    rvm install ruby
    ruby -v
  5. Ruby installs the contents for all the packages. To inhibit this, type the following command:
    sudo echo "gem: --no-ri --no-rdoc" > ~/.gemrc
  6. Update all package of operating system and prevent any dependencies of Rails libraries, install nodejs by typing:
    sudo apt-get update
    sudo add-apt-repository ppa:chris-lea/node.js
    sudo apt-get install nodejs

Install common and useful gems

Bundler is a tool to manage gems while developing and running Rails applications. Nokogiri is a dependency for many other gems.

  1. Update the gem manager.
    gem update --system
  2. Type the following command to use Ruby with gemset global:
    rvm gemset use global
  3. Install both of the gems.
    gem install bundler
    gem install nokogiri

Install Rails

  1. Use gem to install Rails. This will take some time to install.
    gem install rails
    To check the version, type:
    rails -v
  2. Create the project directory for your Rail project, by typing the following commands:
    sudo mkdir -p /usr/local/src/rail_project_directory
  3. Move to the directory just created, and grant all the required access using the following commands:
    cd /usr/local/src/rail_project_directory
    sudo chmod -R 777 /usr/local/src/rail_project_directory
    rails new Your_application
    cd Your_application
    rails server

    Now your Rails server should be running. Press CTRL+C to terminate the process.

  4. You can bind your Rails application using the following command:
    rails server --binding=your public IP

    Your new application is ready and can be accessed by visiting your_public_IP_or_domain:3000. 3000 is the default port. Press CTRL+C to terminate the process.

Next steps
