Install Ruby on Rails using RVM - Debian

Difficulty: 1

Time: 30 minutes

Ruby on Rails is a well-known open source stack for web application development. The Ruby Version Manager (RVM) lets you easily manage multiple Ruby application environments and switch between them in the most efficient way. This article will assist on how to install Ruby on Rails in Debain VPS using RVM.

To get started, we will need the following prerequisites:

  • RVM (Ruby Version Manager)
  • SQLite
  • MySQL (Optional)
  • Apache2 (Optional)
  1. To install RVM, run the following commands:
    sudo apt-get install libgdbm-dev libncurses5-dev automake libtool bison libffi-dev
    sudo curl -L https://get.rvm.io | bash -s stable
  2. Once we have run the above command, there is one more dependency that needs to be fixed. For that run the following command:
    sudo curl -sSL https://rvm.io/mpapis.asc | gpg --import -
    sudo curl -L https://get.rvm.io | bash -s stable
    source ~/.rvm/scripts/rvm
    rvm install ruby
    ruby -v
  3. Ruby tends to install the contents for all the packages. So to inhibit this, type the following command:
    sudo echo "gem: --no-ri --no-rdoc" > ~/.gemrc
  4. Then install the bundler
    gem install bundler
  5. To prevent any dependencies of Rails libraries, let us install NodeJS by typing:
    sudo add-apt-repository ppa:chris-lea/node.js
    sudo apt-get update
    sudo apt-get install nodejs
  6. Install Rails
    gem install rails

    This will take some time. Once completed, the environment is created.

  7. To check you rails version, type:
    rails -v
  8. Create the project directory for your Rail project by typing the following command:
    sudo mkdir â€“p /usr/local/src/rail_project
  9. Move to the newly created directory and grant all the required access using the following commands:
    cd /usr/local/src/rail_project
    sudo chmod 777 /usr/local/src/rail_project
    rails new HelloWorld
    cd HelloWorld
    rails server

    Ensure the Rails server is running.

  10. You can bind the Rails application using the following command:
    rails server --binding=[your public IP]

The new application is ready and can be accessed by visiting your_public_IP_or_domain:3000 (3000 is default port).
