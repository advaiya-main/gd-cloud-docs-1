Install Java

Install Java with yum - CentOS, Fedora

Difficulty: 1
Time: 10 minutes

Java is used by many applications, so benefits many administrators to have Java already installed. This article contains directions for installing the java runtime, SDK, and Open JDK 7 versions.

Install Java

  1. Update your repository.
    sudo yum update
  2. Download Java runtime. This article uses version 1.7.0. Update the version number based on the current version of Java.
    sudo yum install java-1.7.0-openjdk
    Enter y at the prompt to continue installation.
  3. Download Java OPEN JDK 7.
    sudo yum install java-1.7.0-openjdk-devel
    Enter y at the prompt to continue installation.
  4. You can also get the latest version number and tar.gz archive from the Oracle downloads page. This site requires a license agreement.

Select and verify the default Java version

  1. If you have more than one version of Java installed, you can use the following command to select it. Unless you installed an Oracle version above, this command will only return one row.
    sudo update-alternatives --config java
    You will see this: 
    Selection   Command
-----------------------------------------------
*+ 1      /usr/lib/jvm/java-1.7.0-openjdk-1.7.0.79-2.5.5.0.fc20.x86_64jre/bin/java

Enter to keep the current selection[+], or type selection number:
  2. Create a /etc/environment file.
    sudo vim /etc/environment
  3. To select your preferred version, use the JAVA_HOME command and the path of your preferred alternative with the jre/bin/java part of the path removed. Enter this path into the file.
    JAVA_HOME="/usr/lib/jvm/java-1.7.0-openjdk-1.7.0.79-2.5.5.0.fc20.x86_64"
  4. Save and close.
  5. Load the environment file.
    source /etc/environment
  6. Test it.
    echo $JAVA_HOME

Conclusion

You have just installed three versions of Java, and then selected one to be your default.
