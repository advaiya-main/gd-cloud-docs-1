Install and configure Concrete5 on your LAMP server – CentOS, Fedora

Difficulty: 1
Time: 30 minutes

Introduction

Concrete5 is an open source Content Management System (CMS) made in PHP. Concrete5 is easy to understand and learn.

In this article, you will learn how to install and configure Concrete5 in CentOS and Fedora.

Prerequisites

  1. Ubuntu
  2. LAMP Install and configure LAMP Ubuntu
  3. sudo user

Install required dependency

  1. Install the unzip tool which is required to extract file.
    sudo yum install unzip
  2. Install php libraries for Graphics and xml.
    sudo apt-get install php-gd php-xml
  3. Restart the apache services for changes to take effect.
    sudo service restart httpd

Basic configuration

  1. Run the following command to download Concrete5 source from official site:
    cd /var/www/html
    sudo wget --trust-server-names http://www.concrete5.org/download_file/-/view/58379/8497/ -O concrete5.zip
  2. Extract downloaded source using the following command:
    sudo unzip concrete5.zip
  3. Rename file to short name. The new directory is added in folder namely ‘concrete5.6.2.1’.
    sudo mv concrete5.6.2.1 concrete5
  4. Change Directory to concrete5.
    cd concrete5

Create database

  1. Create blank database for backend operation.
    mysql –u root –p
    CREATE DATABASE concrete5;
  2. Exit.

Install Concrete5

  1. Change permission and group of following to make writable for concrete5.
    sudo chown -R root:apache files/
    sudo chown -R root:apache config/
    sudo chown -R root:apache packages/

    Change permission of following folders.

    sudo chmod -R 775 files/
    sudo chmod -R 775 config/
    sudo chmod -R 775 packages/
  2. Open browser and type public IP or URL of concrete5 server. http://<server_ip>/concrete5
  3. Hit Enter to see the first step of concrete5 installation process.
  4. Press on continue to install concrete5.
  5. Enter settings and text for site configuration concrete5.
  6. Click on Install Concrete5. The installation process screen will show up.
  7. The Installation process is complete.
  8. Press on continue site, you will be redirected to Welcome screen of concrete5.
  9. Select process as per your requirement.

Conclusion

In this article, you have learnt to install and configure Concrete5 in CentOS and Fedora.
