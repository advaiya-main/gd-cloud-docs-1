Install and configure Concrete5 on your LAMP server – CentOS, Fedora

Difficulty: 1
Time: 30 minutes

Concrete5 is an open source Content Management System (CMS) made in PHP. Concrete5 is easy to understand and learn. In this article, you will learn how to install and configure Concrete5 on CentOS and Fedora.

Before you proceed, you must have access to sudo priviledges to perform the steps mentioned in this article. In addition, you will need to build a LAMP stack by referring the following links:

  • Build a LAMP stack (Linux, Apache, MySQL, PHP) - CentOS 6
  • Build a LAMP stack (Linux, Apache, MySQL, PHP) - CentOS 7
  • Build a LAMP stack (Linux, Apache, MySQL, PHP) - Fedora

Install the required dependency

  1. Install the unzip tool which is required to extract the file.
    sudo yum install unzip
  2. Install the PHP libraries for graphics and XML.
    sudo apt-get install php-gd php-xml
  3. Restart the Apache services to make the changes effective.
    sudo service restart httpd

Basic configuration

  1. Run the following command to download the source of Concrete5 from the official site:
    cd /var/www/html
    sudo wget --trust-server-names http://www.concrete5.org/download_file/-/view/58379/8497/ -O concrete5.zip
  2. Extract the downloaded source using the following command:
    sudo unzip concrete5.zip
  3. Rename the file concrete5.6.2.1 as concrete5 to give it a shorter name.
    sudo mv concrete5.6.2.1 concrete5
  4. Change directory to concrete5.
    cd concrete5

Create database

  1. Create a blank database for backend operation.
    mysql –u root –p
    CREATE DATABASE concrete5;
  2. Exit.

Install Concrete5

  1. Change permissions of the following to make them writable for concrete5.
    sudo chown -R root:apache files/
    sudo chown -R root:apache config/
    sudo chown -R root:apache packages/

    Change permissions of the following folders.

    sudo chmod -R 775 files/
    sudo chmod -R 775 config/
    sudo chmod -R 775 packages/
  2. Open browser and type public IP or URL of your concrete5 server as http://<server_ip>/concrete5 and then press the Enter key.
  3. On the installation page, click on Continue to Installation.
  4. Enter the required information for Site, Database and Administrator and then click on the Install Concrete5 button.
  5. After the installation process is complete, click on the Continue to your site button.

    6. You will be redirected to the Welcome screen of concrete5.


  6. Select any of the guide as per your requirement.

Conclusion

In this article, you have learnt to install and configure Concrete5 in CentOS and Fedora.
