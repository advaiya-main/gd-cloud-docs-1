Setting up Ruby on Rails using RVM in Arch Linux, Ruby, Rails

Install Ruby on Rails using RVM - Arch Linux

Difficulty: 2
Time: 20 minutes

Ruby on Rails™ is a well-known framework for web application development. The Ruby Version Manager (RVM) allows developers to easily manage multiple Ruby application environments and switch between them in an efficient way.

This article walks through how to install Ruby on Rails on Debian using RVM, and assumes that you have sudo privileges. See Initial Server Setup - Arch Linux for more on how to set up your server.

Prepare your system

Before you install Ruby on Rails, prepare your system with the required system software.
  1. Update pacman.
    sudo pacman -Syu
  2. Install curl.
    sudo pacman -S curl
  3. Install tar.
    sudo pacman -S tar

Install RVM

  1. Install GPG signatures
    gpg2 --keyserver hkp://keys.gnupg.net --recv-keys 409B6B1796C275462A1703113804BB82D39DC0E3
  2. Install the RVM dependencies using the following commands:
    sudo curl -sSL https://rvm.io/mpapis.asc | gpg2 --import -
  3. If your system doesn't have RVM already installed, type the following command to install RVM:
    sudo curl -L https://get.rvm.io | bash -s stable
  4. Open .bashrc file.
    sudo vim .bashrc
  5. Add the following code at the end of the file:
    [[ -s "$HOME/.rvm/scripts/rvm" ]] && source "$HOME/.rvm/scripts/rvm"
  6. Save and exit the file by pressing the Esc key, typing :wq, and then pressing the Enter key.
  7. Reload the source to enable rvm command.
    source ~/.rvm/scripts/rvm
  8. Log back in to apply the changes, and then install the dependencies of RVM.
    rvm requirements

Install Ruby

  1. Install the following dependencies of Ruby.
    sudo pacman -S libyaml
  2. Install Ruby with RVM.
    rvm install ruby
    To check ruby version, type:
    ruby -v
  3. The Asset Pipeline in Rails needs JavaScript runtime to compile code. On Arch Linux, it's best to use Node.js (a server-side JavaScript environment).
    sudo pacman -S nodejs

Install RubyGems

  • Install gems using RVM.
    rvm rubygems current

Install Rails

  1. Use gem to install the most recent stable release of Rails.
    gem install rails
    To check the version, type:
    rails -v

Next steps

Congratulations! You've set up Ruby on Rails using RVM for your web application. For more information on RVM, see Use RVM to manage Ruby installations.
