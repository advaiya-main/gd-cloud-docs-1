Installation of Node.js

Install Node JS on your cloud server - Ubuntu, Debian

Difficulty: 1
Time: 10 mins

Node.js is a web platform based entirely in Javascript that can be used to rapidly deploy scalable sites and site networks. With the advent of the Chrome JavaScript V8 runtime engine, Javascript can now be run server-side for deploying web applications. Because JavaScript uses event-driven and non-blocking I/O, Node.js is able to deliver high performance websites with the capability to handle a large amount of concurrent connections with a low resource impact. This article walks through installing Node in 3 ways.

Basic installation

This is the distro-stable version you will be installing. By default, Ubuntu 14.04 and Debian 7 contain the version 0.10.25 of Node.js in its repositories.

  1. As is best practice run update to update the installed packages.
    sudo apt-get update
  2. Install Node.js
    sudo apt-get install nodejs
  3. You can also install the Node.js Package Manager (NPM) to manage the modules and packages.
    sudo apt-get install npm
  4. You can verify the the current verion of Node.js that got installed by :
    nodejs -v

PPA Installation

    You can also install Node.js using a PPA (Personal Package Archive). Compared to the default repositories, this has more up-to date versions of Node.js.
  1. Update the installed packages.
    sudo apt-get update
  2. Install PPA to get access to its contents.
    curl -sL https://deb.nodesource.com/setup | sudo bash -
  3. Install the node.js
    sudo apt-get install nodejs
  4. The nodejs package already contains the pakage manager npm, so you don't have to install npm like you did in the previous section. But it you would need to install another package build-essentials for some of the npm packages to work.
    sudo apt-get install build-essential
  5. You can verify the the current verion of Node.js that got installed by :
    node -v

NVM installation

The third method to install Node.js is through the Node Version Manager (NVM) which essentially allows you to manage different versions of Node.js in the same box.You will be able to select the any version of Node.js from older right till the latest version.
  1. Update the repositories.
    sudo apt-get update
  2. Install the build-essential package that is required to build your components with nvm.
    sudo apt-get install build-essential libssl-dev
  3. Install nvm installation script from the Github page
    curl https://raw.githubusercontent.com/creationix/nvm/v0.26.1/install.sh | sh

    In this guide we are installing version 26. You can choose to install any version instead of it.

    This will install it to a subdirectory of your home directory at ~/.nvm along with adding code to ~./profile

  4. Source the ~/.profile for your session to accept the changes, so that you now can access the NVM functionality.
    source ~/.profile

Basic nvm commands

Once nvm is installed you can choose to install any version of Node.js that is available. Listed below are some basic commands that you can use with nvm.

  • To see the list of available versions:
    nvm ls-remote
  • If you wish to install a version from the available list, you can do so with the below command. For this example we will install version 0.12.7.
    nvm install 0.12.7
  • Configure nvm to use the downloaded version.
    nvm use 0.12.7
  • You can see the current version of Node.js that is installed with :
    node -v
  • If you have several versions of Node.js installed, you can check the installed version with :
    nvm ls
  • In case you have several versions, you can default one of those by :
    nvm alias default 0.12.7
    You can also reference it by :
    nvm use default
  • You can use NPM to install the packages to Node.js project's ./node_modules directory. For example we will install the express package. Express is a minimal but flexible HTTP server framework.
    npm install express
  • To install it globally so that other projects using the same verson of Node.js, use the -g option.
    npm install -g express
    The package will be installed in ~/.nvm/versions/node/v0.12.7/lib/node_modules/express
  • Now that you have installed it globally, you will be able to run the commands from the command line. Before you do that you need to link the package into your local sphere.
    npm link express
  • For more options, use:
    nvm help

Next steps

Congratulations! You've just installed Node.js with its NPM system.
