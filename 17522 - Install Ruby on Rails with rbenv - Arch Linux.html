Setting up Ruby on Rails using rbenv on Arch Linux, Ruby, Rails

Install Ruby on Rails Using rbenv - Arch Linux

Difficulty: 1

Time: 30 minutes

Ruby is a pure object oriented language like Perl and Python. Rails is a framework for web application development written in Ruby. In this article, you'll learn to install Ruby on Rails on Arch Linux, with the help of rbenv, a lightweight Ruby version management tool.

To set up the Arch Linux server, refer to Initial Server Setup - Arch Linux. Optionally, you can also install Git.

Please make sure that you have assigned super user privileges to your user.

Install rbenv

  1. Upgrade and install all the latest dependencies.
    sudo pacman -Syu
    sudo pacman -Sy base-devel
  2. Install rbenv and Ruby dependencies.
    sudo pacman -Sy git zsh
    git clone git://github.com/sstephenson/rbenv.git ~/.rbenv
  3. Install rbenv, and check that rbenv is located in .rbenv directory.
    cd ~/.rbenv
    echo 'export PATH="$HOME/.rbenv/bin:$PATH"' >> ~/.bash_profile
    sudo echo 'eval "$(rbenv init -)"' >> ~/.bash_profile
    exec $SHELL
    source ~/.bash_profile
  4. Install the ruby-build plugin.
    5. git clone git://github.com/sstephenson/ruby-build.git ~/.rbenv/plugins/ruby-build

Install Ruby

  1. Install the Ruby using the following command:
    rbenv install -v 2.2.3
  2. Set Ruby as default to use. Set the default version of Ruby that all of your shells will use.
    rbenv global 2.2.3
  3. Display which version of Ruby was installed.
    ruby -v

Install bundler

  1. Disable document installation. (By default, when you install gems, documentation files are also installed. But this takes time, so let's stop the document installation.)
    echo "gem: --no-document" > ~/.gemrc
  2. Install the bundler gem which is an essential tool for managing gems when developing and running Rails applications.
    gem install bundler

Set up Rails

  1. Install the Rails.
    gem install rails

    This installation takes some time.

  2. Install shims for all Ruby executables.
    rbenv rehash
  3. Check the version of both Ruby and Rails to make sure that they installed.
    ruby -v
    rails -v

Install JavaScript runtime

A few Rails features, such as Asset Pipeline, depend on a JavaScript runtime.

  • Install the Node.js package.
    sudo pacman -S nodejs npm
