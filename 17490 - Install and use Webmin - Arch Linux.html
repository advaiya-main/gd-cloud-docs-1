Install Webmin - Arch Linux

Difficulty: 2
Time: 15 minutes

Webmin is a web-based tool for managing your server. It supports many common tasks like installing software, adding users, and managing servers. This article describes how to install Webmin. It assumes you have a user, yourloginuser, who accesses your server via SSH.

Get Webmin

  1. Change to the tmp directory.
    cd /tmp
  2. Download Webmin.
    sudo wget http://prdownloads.sourceforge.net/webadmin/webmin-1.760.tar.gz
  3. Unzip it.
    sudo gunzip webmin-1.760.tar.gz
  4. Unzip the tar archive.
    sudo tar -xf webmin-1.760.tar
  5. Navigate to the directory created by tar.
    cd webmin-1.760
  6. Install Webmin.
    sudo ./setup.sh /usr/local/webmin

    The install program will ask several questions. Accept the defaults by pressing Enter on your keyboard at each one, until you get to the Login name prompt. Create a login name and password. Enter y at the last prompt.

Verification

  1. Using your browser, access the Webmin login page: https://Your IP address:10000
  2. A login screen will appear. Log in using yourloginuser, and start using Webmin!
