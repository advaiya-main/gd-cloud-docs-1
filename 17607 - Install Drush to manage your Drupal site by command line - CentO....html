Setting up Drush, Drupal, PHP, Applications

Install Drush to manage your Drupal site by command line - Debian, Ubuntu

Difficulty: 1
Time: 1 hour

Drush is a command-line interface that manages a Drupal framework. When compared to the user interface (UI) of Drupal, Drush provides a much faster way to manage Drupal sites by eliminating various time-consuming tasks.

In this article, you will learn how to install Drush in Arch Linux.

You need access to sudo privileges to complete these steps.

Confirm that you have LAMP installed on your server. If needed, you can refer to this article: Build a LAMP stack (Linux, Apache, MySQL, PHP) – CentOS 7 and Build a LAMP stack (Linux, Apache, MySQL, PHP) - Fedora.

Install server requirements for Drupal

  1. Update the packages.
    sudo yum update
  2. Install Unzip.
    sudo yum install unzip
  3. Install the php-gd graphic library.
    sudo yum install php-gd phpunit tar
  4. Make the following corresponding changes to the Directory block of /etc/httpd/conf/httpd.conf file to let .htaccess file override the default Apache instructions.
    sudo vim /etc/httpd/conf/httpd.conf

    Updated Directory block should look like:

    <Directory /var/www/html>
		Options Indexes FollowSymLinks MultiViews
		AllowOverride All
		Order allow,deny
		allow from all
</Directory>
  5. Save and exit file by pressing Esc key and then typing :wq followed by the Enter key.
  6. Restart Apache.
    sudo service httpd restart

Download Drush files

  • Download Drush using the following command.
    sudo wget https://github.com/drush-ops/drush/archive/master.zip

Install Drush

  1. Unzip Drush file archive.
    sudo unzip -q master.zip
  2. Rename ~/drush-master directory for ease of use.
    sudo mv ./drush-master ./drush
  3. Add alias drush='~/drush/drush' into .bashrc file using the following command to let bash executable recognize the alias drush.
    echo "alias drush='~/drush/drush'" >> .bashrc

Install composer

  1. Get ownership of drush directory.
    sudo chown –R user_name:user_name drush/
  2. Change the directory to drush.
    cd drush
  3. Get composer files and pipe them to PHP for execution.
    sudo curl -sS https://getcomposer.org/installer | php
  4. Install Drush using the following command.
    sudo php composer.phar install

Cleanup

  1. Navigate to the root directory.
    cd ~
  2. Reset bash settings.
    source .bashrc
  3. Confirm the drush installation with following command. This will display the help options of Drush.
    drush help

Create a MySQL database and user

  1. Log into the MySQL.
    mysql -u root –p

    It will prompt you for the root password.

  2. Create a database with the following command.
    CREATE DATABASE drupal;
  3. Create a user.
    CREATE USER drupal_user@localhost;
  4. Set password for the new user.
    SET PASSWORD FOR drupal_user@localhost= PASSWORD("password ");
  5. Grant privileges.
    GRANT ALL PRIVILEGES ON drupal.* TO drupal_user@localhost IDENTIFIED BY 'password ';
  6. Refresh MySQL.
    FLUSH PRIVILEGES;
  7. Exit from the MySQL shell.
    exit

Download and Configure Drupal

  1. Change ownership of the main html folder.
    sudo chown -R user_name:user_name /var/www/html/
  2. Navigate to the root folder.
    cd /var/www/html
  3. Download Drupal. You can give any name to your Drupal project/site.
    drush dl drupal --drupal-project-rename=drupal_site
  4. Run the following command to configure Drupal and change parameters as per your need.
    drush site-install standard --db-url=mysql://drupal_user:password@localhost/db_name --site-name=your_site_name --account-name=admin --account-pass=your_password

    It will create a user with username account-name and given password.

  5. Assign the sites/default/ group ownership to the apache group.
    sudo chown -R root:apache sites/default/files
  6. Change permissions of sites/default/files for apache group.
    sudo chmod -R 775 sites/default/files
  7. Access your Drupal installation and give admin credential that you have created.
    http://server_ip /drupal_site /

Conclusion

In this article, you have learnt to install Drush on your CentOS and Fedora server. You can easily manage your site via the command line using Drush. To execute drush commands, you need to be on the document root of your Drupal site.
