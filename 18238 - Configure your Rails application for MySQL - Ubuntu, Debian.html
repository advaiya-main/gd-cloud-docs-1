Setting up Rails application for MySQL server on Debian or Ubuntu, MySQL, Ruby, Rails.

Configure your Rails application for MySQL – Debian, Ubuntu

Difficulty: 1
Time: 30 minutes

This article will assist you to install MySQL, install MySQL adaptor, create a Rails Application, configure database connection and finally create an application database on Debain or Ubuntu distribution.

We assume that you are logged in as a sudo user and installed Ruby on Rails using RVM on debian and installed Ruby on Rails using RVM on Ubuntu links.

Install MySQL

  1. Update all the packages of the operating system.
    sudo apt-get update
  2. To Install MySQL and all its dependencies, type :
    sudo apt-get install mysql-server mysql-client libmysqlclient-dev

    During the installation, your server will ask you to select and confirm a password for the MySQL "root" user.

    When the installation is complete, we need to run some additional commands to get our MySQL environment set up securely. First, we need to tell MySQL to create its database directory structure where it will store its information. You can do this type:

    sudo mysql_install_db
  3. Run the MySQL installation process,type:
    sudo mysql_secure_installation
  4. After you have finished the above command, a welcome screen displays the following 
    	NOTE: RUNNING ALL PARTS OF THIS SCRIPT IS RECOMMENDED FOR ALL MySQL
	SERVERS IN PRODUCTION USE!  PLEASE READ EACH STEP CAREFULLY!


	In order to log into MySQL to secure it, we'll need the current
	password for the root user.  If you've just installed MySQL, and
	you haven't set the root password yet, the password will be blank,
	so you should just press enter here.
	
	Enter current password for root (enter for none):

    If you have created a password by using the following command, then Press Enter.

    /usr/bin/mysqladmin -u root password 'new-password'
    /usr/bin/mysqladmin -u root -h myt password 'new-password'

    Next, answer the following questions.

    	  Set root password?                     --> Y
	  Remove anonymous users?                --> Y
	  Disallow root login remotely?          --> Y
	  Remove test database and access to it? --> Y
	  Reload privilege tables now?           --> Y

Install MySQL adapter

  1. Before connecting Rails application to MySQL Server, we need a MySQL Adapter. mysql12 gem provides this functionality. To install it using the following command:
    sudo gem install mysql2
  2. To update gem version, type:
    sudo gem update –system

Set up your Rails application

  1. Install Rails using following command:
    gem install rails
  2. Create a Rails application and Use the -d mysql option to set MySQL as the database.
    rails new Your_application -d mysql
  3. Go to the application directory.
    cd Your_application

Set up database connection

If you have MySQL's root user password, then the MySQL root login will be used to create your application's test and development databases.

  1. Open your application database configuration file.
    sudo vim config/database.yml
  2. In this file type following in the password section:
    password: mysql_root_password.
  3. Save and exit the file by presssing the Esc key, and then typing :wq, following by the Enter key.
  4. Connect to the database using the MySQL client.
    mysql -u root –p
    Enter the password for mysql_root_password.

Create a new MySQL database

  1. Create a new database using following command:
    create database MyDatabase_name;
  2. Revoke the privileges.
    FLUSH PRIVILEGES;
  3. Exit from Database.
    exit

Conclusion

By using all the above steps you will be able to configure MySQL Server on Debian or Ubuntu for your Rails application.
