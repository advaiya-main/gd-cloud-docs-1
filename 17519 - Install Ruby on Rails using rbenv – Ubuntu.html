Setting up Ruby on Rails using rbenv on Ubuntu, Ruby, Rails.

Install Ruby on Rails using rbenv – Ubuntu

Difficulty: 2

Time: 30 minutes

Ruby is a pure object oriented language like PERL and Python while Rails is a web application development framework written in Ruby. This article will assist you to install Ruby on Rails on Ubuntu, with the help of rbenv. rbenv is a new lightweight Ruby version management tool.

Make sure that you are logged in as a sudo user.

Install rbenv

  1. Update all the packages of operating system.
    sudo apt-get update
  2. Install the rbenv and all Ruby dependencies.
    sudo apt-get install git-core curl zlib1g-dev build-essential libssl-dev libreadline-dev libyaml-dev libsqlite3-dev sqlite3 libxml2-dev libxslt1-dev libcurl4-openssl-dev python-software-properties libffi-dev
  3. To install rbenv, run the following command to check rbenv into .rbenv directory.
    cd
    git clone git://github.com/sstephenson/rbenv.git .rbenv
  4. Add a directory to your $PATH for accessing the rbenv.
    echo 'export PATH="$HOME/.rbenv/bin:$PATH"' >> ~/.bash_profile
  5. Add rbenv init into the shell to enable shims.
    echo 'eval "$(rbenv init -)"' >> ~/.bash_profile
    exec $SHELL
  6. This command will install the latest development version of ruby-build into the ~/.rbenv/plugins/ruby-build directory.
    git clone git://github.com/sstephenson/ruby-build.git~/.rbenv/plugins/ruby-build
  7. Add a directory to your $PATH for accessing ruby.
    echo 'export PATH="$HOME/.rbenv/plugins/ruby-build/bin:$PATH"' >> ~/.bash_profile
    exec $SHELL
  8. This will install rbenv into your home directory, and set the appropriate environment variables that will allow rbenv to the active version of Ruby.

Install Ruby

  1. Install the dependency of Rails, before you install Ruby by typing:
    rbenv install -v latest_version of Ruby
  2. This command will set the default version of Ruby that all of your shells will use.
    rbenv global latest_version of Ruby
  3. To check the version of Ruby installed, type:
    ruby -v

Install RubyGems

  1. By default, when we install gems, documentation files will also get installed along with it. However, this takes time. So here you need to first stop the document installation by disabling them.
    echo "gem: --no-document" > ~/.gemrc
  2. After you have disabled the document installation, install the bundler gem. The Bundler gem is an essential tool for managing gems when developing and running Rails applications.
    gem install bundler

Set up Rails application

  1. To install Rails, type:
    gem install rails

    This will take some time, so be patience. Once completed, Ruby on Rails will be installed.

  2. To install shims for all Ruby executables known to rbenv, use the rehash command.
    rbenv rehash
  3. To check the version of both Ruby and Rails, type:
    ruby -v
    rails –v

Install the JavaScript Runtime

  1. Add the node.js PPA to apt-get:
    sudo add-apt-repository ppa:chris-lea/node.js
  2. Install the node.js package.
    sudo apt-get install nodejs

Conclusion

By using the above steps, you will be able to install rbenv, Ruby, RubyGems and Rails application on Ubuntu.
