Install and configure Drupal on your LAMP server - Arch Linux

Drupal is one of the popular and open-source dynamic CMS for your blogs and website. Drupal is a free software package that allows you to easily organize, manage and publish your content, with an endless variety of customization.

Prerequisites

  1. LAMP

Set up environment for Drupal

  1. Install the php graphics, libssh and php-curl which will required by Drupal in future.
    sudo pacman -Syu update
    sudo yum install php-gd
  2. Uncomment following line in php.ini file.
    sudo nano /etc/php/php.ini
    extension=pdo_mysql.so
    extension=mysql.so
    extension=mcrypt.so
    extension=gd.so
  3. Reload apache2 webserver to change effect.
    sudo systemctl restart httpd.service
  4. Create database for Drupal to store basic settings and user data.
    mysql -uroot -ppassword
    CREATE DATABASE drupal;
  5. Add new user for Drupal database.
    CREATE USER drupaluser@localhost IDENTIFIED BY 'drupalpassword';
  6. Grant all privileges to the drupaluser.
    GRANT SELECT,INSERT,UPDATE,DELETE,CREATE,DROP,INDEX,ALTER,CREATE TEMPORARY TABLES,LOCK TABLES ON drupal.* TO drupaluser@localhost;

    OR

    GRANT ALL PRIVILEGES ON drupal.* TO 'drupaluser'@'localhost' IDENTIFIED BY 'drupalpassword' WITH GRANT OPTION;
  7. Reload the MySQL database privileges to activate the new MySQL user.
    FLUSH PRIVILEGES;
    exit

Set up Drupal

  1. Download the Drupal stable version from the official site using the following command:
    cd /srv/http
    sudo wget http://ftp.drupal.org/files/projects/drupal-7.39.tar.gz
  2. Extract Drupal compressed file using the following command:
    sudo tar zxvf drupal-7.39.tar.gz
  3. Rename Drupal extracted folder to 'drupal' for easy access.
    sudo mv drupal-7.39 drupal
  4. Change permission to 755 to the Drupal directory.
    sudo chmod -R 755 /srv/http/drupal
    sudo chown -R http:http /srv/http/drupal

Install Drupal

  1. Open the browser and type FQDN or Public IP in browser to start installation for drupal for your blog.
    http://public_ip/drupal
  2. Select an installation type as per your requirement.

  3. Choose language as per your need.

  4. In Database configuration, type the correct information of database. In Advance Options, give Connection host, Port and Table prefix.

  5. Configure your blog and website information in the Configuration site page. Type the appropriate information, and click Save and continue.

    This process may take some time.

  6. It will show you the Drupal installation complete page. Click on Visit your new site to go to your Drupal website.

  7. You will see a welcome page for Drupal Blog. You can start blogging here.

Conclusion

In this article you learned to install and configure Drupal on your LAMP server with ArchLinux.
