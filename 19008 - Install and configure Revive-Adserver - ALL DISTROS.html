Install and configure Revive-Adserver - ALL DISTROS

Difficulty: 2
Time: 20 minutes

Revive-Adserver is a php-based advertisement server that allows you to track and manage ad campaigns. This article assumes you have tar and a LEMP stack installed appropriate to your Linux distribution:

  • Arch Linux
  • Centos 6
  • Centos 7
  • Debian
  • Fedora 20
  • Fedora 21
  • Ubuntu

Install Revive-Adserver

  1. Create a directory for your Revive-Adserver installation under your nginx root (usually /usr/share/nginx/html or /usr/share/nginx/www
    sudo mkdir /usr/share/nginx/html/revive-adserver
  2. Download Revive-Adserver
    sudo wget http://download.revive-adserver.com/revive-adserver-3.2.1.tar.gz
  3. Unzip it
    sudo tar xf revive-adserver-3.2.1.tar.gz 
  4. Move it to revive-adserver
    mv revive-adserver-3.2.1/* /usr/share/nginx/html/revive-adserver/

Create Revive-Adserver database

  1. Open a mysql session
    mysql -u root -p
    Enter the mysql root pasword when prompted. A mysql prompt (>) will display
  2. Create the revive-adserver database
    mysql> create database revive;
  3. Navigate to your newly created database
    mysql> use revive;
  4. Create a user
    mysql> create user 'revive'@'localhost' identified by 'Password';
  5. Grant privileges
    mysql> grant all on revive.* to 'revive'@'localhost';
  6. Write changes to disk
    mysql> flush privileges;
  7. Exit Mysql
    mysql> exit

Verify Installation

Using your browser, navigate to http://your server name or IP/revive-adserver. You will see the revive-adserver UI. Follow the instructions to register and complete your set up.

Next steps

Use the revive UI to set up your adserver as you see fit, keep in mind that the database it refers to is the database you created in mysql, above..
