Install and configure a Riak database cluster - Ubuntu

Difficulty: 2
Time: 15 minutes

In this article, you will learn how to install Riak database cluster on an Ubuntu Droplets. Riak is a distributed database system which is designed for high availability of data.

Ubuntu servers with 4 GB RAM is preferred for better performance. Here you will perform all installations as a root user.

Install Riak

Perform the following installation steps on all nodes which are included in this Riak cluster.

  1. Execute below commands to add Riak repository and key.
    curl -s https://packagecloud.io/install/repositories/basho/riak/script.deb.sh | sudo bash
  2. Install Riak.

    Update apt repository and install Riak.

    apt-get update
    apt-get install riak

Configure first node

Configure first node in Riak cluster with following steps.

  1. Stop Riak service if it was started earlier.
    riak stop
  2. Edit Riak configuration file.

    Find and replace below lines in /etc/riak/riak.conf .

    listener.protobuf.internal = node_ip_address:8087
    listener.http.internal = node_ ip_address:8098
    nodename = riak@node_ip_address
  3. Start Riak service.

    While starting, configure ulimit to minimum of 65565.

    ulimit -u 65565
    riak start

    In the same way, you can configure all other nodes which you want to include in the cluster.

Add second node to cluster

Perform all the steps mentioned above to start Riak in second nodes and on the following nodes which you wish to include in cluster.

  1. Join to first node.

    Inside second node, execute below command to join to first node.

    riak-admin cluster join riak@first_node_ip
  2. Execute below commands to plan and commit the changes:
    riak-admin cluster plan
    riak-admin cluster commit
  3. View the complete ring-cluster by executing the below command.
    riak-admin status | grep ring_members
  4. Set the ulimit value to persist after reboot.

    Create file /etc/default/riak and add below line to make ulimit setting permanent.

    vi /etc/default/riak
    ulimit -n 65536

    Save and close the file.

Conclusion

In this article, you have learnt to install multiple nodes, configure them to start Riak and then join them to first node to build a Riak database cluster.
