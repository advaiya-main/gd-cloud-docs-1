Install Drush to manage your Drupal site by command line - Arch Linux

Difficulty: 1

Time: 1 hour

Drush is a command-line interface that manages Drupal framework. When compared to user interface (UI) of Drupal, Drush provides a much faster way of managing Drupal sites by eliminating various time-consuming tasks.

In this article, you will learn how to install Drush in Arch Linux.

You need access to sudo privileges to complete these steps.

Confirm that you have LAMP installed on your server. You can refer to this article Build a LAMP stack (Linux, Apache, MySQL, PHP) - Arch Linux

Install server requirements for Drupal

  1. Update the packages of operating system.
    sudo pacman -Syu
  2. Install Unzip
    sudo pacman –Su unzip
  3. Install php-gd graphic library.
    sudo pacman -Su php-gd tar
  4. Open php.ini file in a text editor.
    sudo vim /etc/php/php.ini
  5. Uncomment the following lines from Dynamic Extensions section by removing the semicolon. 
    extension=phar.so
extension=openssl.so
extension=pdo_mysql.so
extension=gd.so
  6. Locate open_basedir and append :/root/ to this line. The final updated line should look like:
    open_basedir = /srv/http/:/home/:/tmp/:/usr/share/pear/:/usr/share/webapps/:/root/
  7. Save and exit the file by pressing Esc, and then typing :wq, followed by the Enter key.
  8. Download phpunit package.
    sudo wget https://phar.phpunit.de/phpunit.phar
  9. Install phpunit package.
    php phpunit.phar
  10. Make the following corresponding changes to <Directory> block of /etc/httpd/conf/httpd.conf file to let .htaccess file override default Apache instructions.
    sudo vim /etc/httpd/conf/httpd.conf

    Updated <Directory> block should look like:

    <Directory /srv/http >
		…
		Options Indexes FollowSymLinks MultiViews
		AllowOverride All
		Order allow,deny
		allow from all
		…
</Directory>
  11. Locate the following line and uncomment it.
    LoadModule rewrite_module modules/mod_rewrite.so
  12. Save and exit the file by pressing Esc, and then typing :wq, followed by the Enter key.
  13. Restart the Apache service to make the changes effective.
    sudo systemctl restart httpd.service

Download Drush files

  • Download Drush using the following command:
    sudo wget https://github.com/drush-ops/drush/archive/master.zip

Install Drush

  1. Unzip Drush file archive.
    sudo unzip -q master.zip
  2. Rename ~/drush-master directory for ease of use.
    sudo mv ./drush-master ./drush
  3. Add alias drush='~/drush/drush' into .bashrc file using following command to let bash executable recognize the alias drush.
    echo "alias drush='~/drush/drush'" >> .bashrc

Install composer

  1. Change ownership of drush directory.
    sudo chown –R user_name:user_name drush/
  2. Change directory to drush.
    cd drush
  3. Get composer files and pipe them to PHP for execution.
    sudo curl -sS https://getcomposer.org/installer | php
  4. Install Drush using the following command:
    sudo php composer.phar install

Cleanup environment

  1. Navigate to the document root directory.
    cd ~
  2. Reset the bash settings.
    source .bashrc
  3. Confirm drush install with following command:
    drush help

    It will display the help options of drush.

Create MySQL database and user

  1. Log into the MySQL.
    mysql -u root –p

    It will prompt you for root password.

  2. Create a database with the following command:
    CREATE DATABASE drupal_Db;
  3. Create a user to login from Drupal site.
    CREATE USER drupal_user@localhost;
  4. Set password for the new user.
    SET PASSWORD FOR drupal_user@localhost= PASSWORD("pass_word");
  5. Grant privileges.
    GRANT ALL PRIVILEGES ON drupal_Db.* TO drupal_user@localhost IDENTIFIED BY 'pass_word';
  6. Refresh MySQL.
    FLUSH PRIVILEGES;
  7. Exit from the MySQL shell.
    exit

Download and configure Drupal

  1. Change the ownership of the main http folder.
    sudo chown -R user_name:user_name /srv/http
  2. Navigate to the document root folder.
    cd /srv/http
  3. Download Drupal. You can give any name to your Drupal project/site.
    drush dl drupal --drupal-project-rename=drupal_site
  4. Change the directory to your druapl site.
    cd drupal_site
  5. Run the following command to configure Drupal and change the parameters as per your need.
    drush site-install standard --db-url=mysql://drupal_user:password@localhost/drupal_Db --site-name=your_site_name --account-name=admin --account-pass=your_password

    It will create a user with username account-name and the given password.

  6. Assign the sites/default/ group ownership to the www-data group.
    sudo chown -R http:http sites/default/files
  7. Change permissions for sites/default/files for www-data group.
    sudo chmod -R 775 sites/default/files
  8. Access your Drupal installation and give the admin credential that you have created.
    http://server_ip/drupal_site/

Conclusion

In this article, you have learnt to install Drush on your system. You can easily manage your site via the command line using Drush. To execute drush commands, you need to be on the document root of your Drupal site.
