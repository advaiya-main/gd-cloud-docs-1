This article provides information about creating a database layer using MySQL to host application servers running Ruby on Rails.

Prerequisites

This article assumes that the host machine has Ruby on Rails using RVM installed and running. For more details, see:
  • Install Ruby on Rails using RVM on Debian
  • Install Ruby on Rails using RVM on Ubuntu
This article assumes that you have configured Rails application for MySQL. For more details, see:
  • Configure your Rails application for MySQL on Debian, Ubuntu

This article assumes that you are logged in as a sudo user.

Install MySQL and other dependencies

  1. Update operating system packages and install MySQL.
    sudo apt-get install mysql-server mysql-client libmysqlclient-dev
    sudo apt-get install build-essential
    sudo apt-get -y update
  2. Initiate the MySQL server daemon by typing:
    sudo /etc/init.d/mysql start
  3. To tell MySQL to create a database directory structure for storing MySQL database directory information, type:
    sudo mysql_install_db
  4. Run the MySQL installation process by typing :
    sudo mysql_secure_installation
  5. After you give the above command, a welcome screen displays the following: 
      NOTE: RUNNING ALL PARTS OF THIS SCRIPT IS RECOMMENDED FOR ALL MySQL 
            SERVERS IN PRODUCTION USE!  PLEASE READ EACH STEP CAREFULLY!


  In order to log into MySQL to secure it, we'll need the current 
      password for the root user.  If you've just installed MySQL, and 
      you haven't set the root password yet, the password will be blank, 
      so you should just press enter here.
  
  Enter current password for root (enter for none):
    Enter the current password for root. If you have not created a root password, create it by using the following command: 
       /usr/bin/mysqladmin -u root password 'new-password'
    Press Enter.

    Answer the following questions by pressing Y.

        Set root password?                     --> Y
    Remove anonymous users?                --> Y
    Disallow root login remotely?          --> Y
    Remove test database and access to it? --> Y
    Reload privilege tables now?           --> Y

Create a database, database user and database connection

  1. Establish a database connection using MySQL client and enter the password for root that was previously set up.
    sudo mysql -u root -p
  2. Create a sample database.
    create database rails_Database_name;
  3. Create a new user for your database and grant remote and local access to this user.
    CREATE USER 'rails_UserName'@'localhost' IDENTIFIED BY 'pwd';
    CREATE USER 'rails_UserName'@'%' IDENTIFIED BY 'pwd';
  4. Grant access rights to any specific user by typing:
    GRANT ALL ON rails_myapp.* TO 'rails_UserName'@'localhost'; GRANT ALL ON rails_myapp.* TO 'rails_UserName'@'%';

    Revoke the privileges.

    FLUSH PRIVILEGES;
  5. Exit from database.
    Exit
  6. Open the MySQL database configuration my.cnf to enable remote connections, and edit using the vim text editor.
    sudo vim /etc/my.cnf
    Type the below mentioned line in this file : 
     
[mysqld]
  ..
  .
bind-address   =  0.0.0.0
  7. Save and exit the file by pressing the Esc key and then typing :wq, followed by the Enter key.
  8. Start MySQL Daemon.
    sudo /etc/init.d/mysql start

Create Rails application

  1. Install gem for Mysql and updates.
    sudo gem install mysql2 sudo gem update --system
  2. Install Rails package with the following command:
    sudo gem install rails
  3. Create a Rails application and use the -d mysql option to set MySQL as the database.
    rails new Your_application -d mysql
  4. Go to the application directory.
    cd rails new Your_application
  5. Go to config directory and open database.yml file.
    cd config/
    sudo vim database.yml
  6. Replace with the following code in production section: 
    production:
  adapter: mysql
  encoding: utf8
  database: rails_Database_name
  username: Your_application 
  password: pwd
  host: 128.199.233.36
  port: 3306
  pool: 10
  7. Save and exit the file by pressing the Esc key and then typing :wq, followed by the Enter key.
    wq!

Install dependencies

  1. Open the Gemfile.
    sudo vim Gemfile
    Add the following line:
    gem 'mysql'
  2. Save and exit the file by pressing the Esc key and then typing :wq, followed by the Enter key.
    wq!
  3. Install all the dependenices of Gemfile.
    bundle install

Conclusion

By using all of the above steps you will be able to scale Ruby on Rails application and set up a dedicated MySQL Server on Debian, Ubuntu.
