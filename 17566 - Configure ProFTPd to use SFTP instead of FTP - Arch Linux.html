Introduction

ProFTPd is an FTP daemon for Linux or UNIX like operating systems. ProFTPd is an open source package to connect FTP securely over the internet.

In this article you will learn, how to install and configure ProFTPd in centOS and Fedora.

Prerequisites

  1. CentOS or Fedora.
  2. Sudo user.

Install Essential Library

  1. Install the C library and make command using the following command:
    sudo pacman -S gcc make

Intall ProFTPd

  1. Download the source ProFTPd using the following the command:
    sudo wget ftp://ftp.proftpd.org/distrib/source/proftpd-1.3.6rc1.tar.gz
  2. Extract the package and change directory to ProFTPd using the following command:
    sudo tar -xf proftpd-1.3.6rc1.tar.gz
    cd proftpd-1.3.6rc1
  3. Use the following command to install ProFTPd.
    sudo ./configure
    sudo make && sudo make install
  4. Install sample configuration file using the following command:
    sudo sh sample-configurations/PFTEST.install
  5. Open the file /tmp/PFTEST/PFTEST.conf in your favorite editor to modify.
    sudo vim /tmp/PFTEST/PFTEST.conf
  6. Set the user and group as nobody and set the document root to access default directory as shown below. 
    User                     nobody
Group                    nobody
DefaultRoot 	        /home/user !adm
  7. Save and exit the file by pressing ESC key and, then type : wq!, followed by the Enter Key.

Configure ProFTPd

  1. Open the PFTEST.conf file to modify.
    sudo vim /tmp/PFTEST/PFTEST.conf
  2. Change the ServerName to your FQDN or your public IP.
    ServerName “Your-public-ip”

    Add the Port below the DefaultServer variable label.

  3. Change the default server port to hide from public scan.
    Port 2021
  4. To start the ProFTPd services, use the following command:

    non-daemon:

    sudo proftpd -n -d 5 -c /tmp/PFTEST/PFTEST.conf

    daemon:

    sudo proftpd -d 5 -c /tmp/PFTEST/PFTEST.conf

Test ProFTPd

  1. Open the filezilla or any other client having SFTP or FTP like WinSCP.
  2. Type your public IP or FQDN in Host textbox.
  3. Enter your user name in the textbox.
  4. Enter your password in the textbox.
  5. Type Port which is typed in the proftpd.conf file, in the textbox which is aside of Port label.
  6. Press the Quickconnect.

Conclusion

In this article, you have learnt to install and configure ProFTPd in centOS and Fedora.
