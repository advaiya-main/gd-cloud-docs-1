Install Virtualmin

Install and use Virtualmin - ALL DISTROS

Difficulty: Value from spreadsheet
Time: Time value

Webmin is a GUI tool that helps you manage the software packages installed on your server. Virtualmin is an extension of Webmin that extends its functionality to managing Virtual Private Servers (VPS).

Install Virtualmin

  1. Download the Virtualmin software
    sudo wget http://software.virtualmin.com/gpl/scripts/install.sh -O /root/virtualmin-install.sh
  2. Your screen will display something like this:
    2015-09-03 13:32:53 (710 KB/s) - ‘/root/virtualmin-install.sh’ saved [47633/47633]
  3. Install Virtualmin
    sudo sh /root/virtualmin-install.sh
  4. You may have logged in to your server using an IP address rather fully qualified domain name (FQDN). The script may prompt you for your FQDN. If this is the case, provide it.
  5. Navigate to your webmin page (MyCoolWebsite.com:10000)
    https://MyCoolWebsite.com:10000

On the left hand side of the page there is a menu. Clicking on the name toggles the UI between Virtualmin and Webmin. You will notice some differences from the standard Webmin interface because the two tools offer slightly different functionality. Webmin manages your server while Virtualmin manages the individual VPS you have installed on your physical server.

Next steps
