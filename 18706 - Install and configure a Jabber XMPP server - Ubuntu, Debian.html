Configuring Jabber XMPP server in Debian and Ubuntu, XMPP

Install and configure a Jabber XMPP server - Debian, Ubuntu

Difficulty: 2

Time: 20 Minutes

Jabber is a simple, mature and powerful messaging protocol based on Extensible Messaging and Presence Protocol (XMPP). It is basically developed developed to collaborate in real-time between two or more people, based on the text typed. Jabber is used by enterprises for internal communication, developers for programming skills improvement, hosting providers for high availability and stable communication services, and by individuals for chatting. Jabber is written in Erlang and has its own web-based management interface. This article explains the installation of Jabber XMPP server on Debian and Ubuntu.

You will need access to sudo privileges to complete all steps.

Install Jabber

  • Use the following command to install the Jabber server along with its components.
    sudo apt-get install ejabberd

After you have installed the server, you need to change a few configuration settings.

Configure Ejabberd

  1. Open ejabberd.cfg file that resides in /etc/ejabberd/.
    sudo vim /etc/ejabberd/ejabberd.cfg
  2. To add the host name, update the following line:
    {hosts, ["localhost"]}

    With this line:

    {hosts, ["localhost","coolexample.com"]}
  3. To create an admin user, update the following line:
    {acl, admin, {user, "", "localhost"}}

    With this line:

    {acl, admin, {user, "my_admin", "coolexample.com"}}
  4. Save and close the file by pressing ESC and then type :wq followed by pressing the Enter key.
  5. Restart the service.
    sudo service ejabberd restart
  6. To set password for the admin user, use the following command:
    sudo ejabberdctl register my_admin coolexample.com password

Conclusion

After following these steps you will now have an overview of the installation and basic configuration of Ejabberd server. To change any other configuration logs into the web-based management panel, you may go to the browser, and type, http://coolexample.com:5280/admin and then enter your admin credentials there.
