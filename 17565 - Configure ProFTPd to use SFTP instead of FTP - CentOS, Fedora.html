Introduction

ProFTPd is an FTP daemon for Linux or UNIX like operating systems. ProFTPd is an open source package to connect FTP securely over the internet.

In this article you will learn, how to install and configure ProFTPd in centOS and Fedora.

Prerequisites

  1. CentOS or Fedora.
  2. Sudo user.

Intall ProFTPd

  1. Install ProFTPd in CentOS or Fedora using the following command:
    sudo yum -y install proftpd
  2. Enable ProFTPd services for boot startup, using the following command:
    sudo systemctl enable proftpd.service
  3. Use the below commands to start, restart and stop the ProFTPd services.
    1. Start PrpFTPd services.
      sudo systemctl start proftpd.service
    2. Stop PrpFTPd services.
      sudo systemctl stop proftpd.service
    3. Restart PrpFTPd services.
      sudo systemctl restart proftpd.service
    4. Check status of PrpFTPd services.
      sudo systemctl status proftpd.service

Configure ProFTPd

  1. Open the proftpd.conf file to modify.
    sudo vim /etc/proftpd.conf
  2. Change the ServerName to your FQDN or your public IP.
    ServerName “Your-public-ip”
  3. Change the default server port to hide from public scan.
    Port 9011

    Add the Port below the DefaultServer variable label.

  4. Restart the ProFTPd services using the following command:
    sudo systemctl restart proftpd.service
  5. Check status service is running or not using the following command:
    sudo systemctl status proftpd.service

    If you found Active (Running) in green color, it means services is running.

Set up firewall rules

Setting up the firewall rules is optional step for those who didn’t configure firewall in CentOS or Fedora Server.

  1. Use the following command to enable incoming request.
    firewall-cmd --add-port=9011/tcp –permanent
  2. Reload the firewall using the below command, to have the added rules effect.
    firewall-cmd –reload
  3. Verify the rule added successful using the following command:
    firewall-cmd --list-ports
  4. Change the SELINUX to allow read/write the files.
    setsebool -P allow_ftpd_full_access=1
  5. Restart the ProFTPd using the following command:
    systemctl restart proftpd.service

Test ProFTPd

  1. Open the filezilla or any other client having SFTP or FTP like WinSCP.
  2. Type your public IP or FQDN in textbox which is aside of Host label.
  3. Type your user name in the textbox which is aside of Username label.
  4. Type your password in the textbox which is aside of Password label.
  5. Type Port which is typed in the proftpd.conf file in the textbox which is aside of Port label.
  6. Press the Quickconnect.

Conclusion
