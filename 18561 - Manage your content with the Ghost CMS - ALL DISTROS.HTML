Manage Your Content with Ghost CMS - ALL DISTROS

Difficulty: 3
Time: 1.5 hrs

Ghost is a powerful blogging platform. Besides simplicity in setup, it has a powerful user interface that lets you manage your content with ease. This article provides an introduction to various content management options provided by Ghost.

This article assumes that the host machine has a Ghost blogging platform deployed and running. For instructions, see one of the following:

  • Install and configure Ghost with NGINX - Ubuntu
  • Install and configure Ghost with NGINX - CentOS, Fedora
  • Install and configure Ghost with NGINX - Arch Linux
  • Install and configure Ghost with NGINX - Debian

Ensure that Ghost is running

  • In a browser, type the IP address or domain name: http://192.169.170.227:2368 or http://coolexample.com.

Start a new blog with Ghost

  1. Open the blog setup page in the browser: http://192.169.170.227:2368/ghost/setup/
  2. Provide the required details, and then click OK, LET'S DO THIS.
  3. After the registration is complete, the Content tab shows the sample preconfigured blog as Welcome to Ghost.

Create a new post

  1. Click NEW POST on the top left corner.
  2. Enter the post title and content in the fields provided. While you enter the content, a preview of the post displays in the PREVIEW window.
  3. Access the help regarding formatting the content via the small icon on the top right corner of the Markdown window (the content editing window). For formatting the content, Ghost uses its own markup language called markdown, which uses simple markdowns for formatting content.

    The Markdown Help window shows options for additional formatting of text, such as bold, emphasize, link, and so on.

    Option for adding tags is available in the bottom left corner.

    Option to publish the draft, save the draft or delete the post are available in the bottom right corner of the window.

    Additional post settings like post URL, publish dates, author name and other meta data can be accessed using the Settings icon, next to the SAVE DRAFT button in the bottom right corner.

Additional blog settings and information

The Settings tab provides additional blog settings:

  • General: General information about the blog such as title, description, logo, cover images, email address, number of posts per page, permanent links with dates and blog themes.
  • Users: List of active users of the blog.
  • Tags: Tags applied across all the articles in the blog.
  • Navigation: Additional navigation options besides Home.
  • Code Injection: Adds external code into the top and bottom of the blog theme, without editing the theme files.
  • Labs: Testing ground for experimental features.
  • About: Information about Ghost version, environment, database, mail, and other information.
Additional help options are available via the question-mark icon, next to the User Name at the top right.

Conclusion

In this article, you learned various ways to manage your content using the Ghost CMS. The Ghost blogging platform provides easy-to-use options to manage the content. Because most options are self-explanatory, you can create, edit, publish and manage posts, without needing additional training.
